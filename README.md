#WALLPAPER SWITCHER SCRIPT FOR NON HYPRLAND SETUPS.
---------------------
Directions for use:
Please put all of your wallpapers in ~/Pictures/Wallpapers/

After that: 
You can either attach a custom keybind or run it through ./wallpaper-switcher.sh

WHEN RUNNING FIRST TIME:
Open your terminal and type:

chmod +x ./wallpaper-switcher.sh
./wallpaper-switcher.sh
